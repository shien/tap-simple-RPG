# フェーズ10: セーブ・統合・仕上げ

## 目的
ローカルセーブ機能、全体統合、最終調整を行う。

## 依存
- フェーズ1〜9すべて

- localStorage を使用
- 生存中のみセーブ保持

- ブラウザ再起動時に復帰可能

- ゲーム全体フローを通しでテスト
- 探索 → 戦闘 → 勝利 → 次マス → ボス → 次エリア
- 死亡 → リスタート

### 調整
- 数値バランス微調整
- モバイルでのタッチ操作確認
- パフォーマンス確認

## テスト
- セーブ/ロードが正しく動作すること
- 死亡時にセーブが消えること
- ブラウザリロードで復帰できること

## 完了条件
- 全機能が統合されて動作する
- セーブ/ロードが動く
- モバイルブラウザで快適に操作できる
