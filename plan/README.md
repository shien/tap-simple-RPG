## 原則

- 各フェーズは独立ブランチで開発
- ロジック層（`src/lib/`）は純粋関数中心にしてテストしやすくする
- UI層（`src/app/`, `src/components/`）はロジック層に依存する形にする
- フェーズ1〜7はロジックのみ（UIなし）、フェーズ8〜9でUI実装
