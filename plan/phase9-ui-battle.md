# フェーズ9: UI — 戦闘画面

## 目的
戦闘画面のUIを実装する。

## 依存
- フェーズ6（戦闘システム）
- フェーズ8（探索画面）

## 実装ファイル
- `src/components/BattleView.tsx` — 戦闘画面コンポーネント
- `src/components/HpBar.tsx` — HPバー
- `src/components/ElementBadge.tsx` — 属性バッジ

## 表示要素
- 敵属性（アイコン+ラベル）
- 武器属性（アイコン+ラベル）
- 相性表示（有利/不利/同属性）
- 倍率表示（×2 / ×0.1 / ×1）
- 敵HPバー（現在HP / MaxHP）
- プレイヤーHP
- 攻撃ボタン
- 異常個体時: 特別表示 + EXP×100表示

## 実装方針
- 攻撃ボタンはタップしやすい大きなボタン
- 敵の自動攻撃はsetIntervalで実装
- 戦闘終了時に結果表示（勝利報酬 or 死亡）

## 完了条件
- 戦闘画面が表示される
- 攻撃タップで敵HPが減少する
- 敵の自動攻撃でプレイヤーHPが減少する
- 属性相性が正しく表示される
- 異常個体の特別表示が出る
